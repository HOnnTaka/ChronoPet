# 更新日志 (Changelog)

## v1.0.3 (2026-02-13)

### ✨ 新特性

- **软件内更新体系**:
  - 实现全自动下载更新功能，检测到新版本后可点击按钮直接在程序内下载。
  - 新增 Win11 风格的**实时下载进度条**，实时反馈下载状态。
  - 下载完成后自动启动安装程序并提示退出软件。
- **Liquid UI 视觉增强**:
  - 优化窗口材质，引入了微妙的 **噪声纹理 (Noise Texture)** 叠加。
  - 增强了 Backdrop Blur (30px) 效果，使得毛玻璃质感更加高级，解决了单纯透明带来的“背景直接混合”感。

### 🐞 修复与优化

- **UI 交互优化**:
  - 精简了发现新版本时的弹窗内容，去掉了冗长的更新日志及描述，使界面更加工整。
  - 修复了进度条在深色模式下的显示一致性。
- **代码健壮性**:
  - 修复了 Dashboard 页面中的语法错误和 Lint 报错。
  - 恢复了 AI 聊天响应的错误处理逻辑。
  - 将安装程序启动方式优化为更安全的 `shell.openPath`。
- **性能**:
  - 优化了主进程与渲染进程间的通信通道，确保更新过程不阻塞主 UI。
